## Headings
You can create headings using the '#' symbol, followed by a space and the heading text. The number of '#' symbols determines the level of the heading. The lower the number, the higher the priority/importance.

Example:
```
# Heading 1
## Heading 2
## Heading 3
```

## Text Formatting
You can apply basic text formatting (bold, italics, strikethrough) using the following Markdown syntax:

- **Bold**: Enclose the text within double asterisks (\*\*text\*\*) or double underscores (\_\_text\_\_).
- *Italics*: Enclose the text within single asterisks (\*text\*) or single underscores (\_text\_).
- ~~Strikethrough~~: Enclose the text within double tildes (\~\~text\~\~).

## Lists

To create ordered lists, use the `-` symbol at the beginning of each line:
```
- Item 1
- Item 2
- Item 3
```

To create ordered lists, use a number followed by a `.`:
```
1. Item 1
2. Item 2
3. Item 3
```

## Links
You can create hyperlinks by placing the link text in square brackets and the URL in parentheses:

```
[Mauro's Technical Documentation source code](https://github.com/maurom88/tech-doc-samples)
```

## Images
To add images: place the alt text in square brackets and the image URL in parentheses.

Example:
```
![A placeholder image](https://picsum.photos/200)
```

## Inline code
You can highlight a piece of code in line by enclosing the code within (\`):

## Code Blocks
You can highlight code snippets or blocks by enclosing the code within triple backticks (\`\`\`). In the following example, remove the backslashes (`\`) to properly use the codeblock:

```
\```
print("Hello, world!")
\```
